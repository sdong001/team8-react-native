# TEAM EIGHT
- 2019-01 ~ ing
- React Native
- 자전거, 이륜차 안전주행을 주제로 BLE 기기와 통신가능한 Android / iOS App을 개발
- 주요 기능
  - 디바이스로 부터 센서 값을 전달받아 사고 인지 후 대응
  - LED pixel art를 서버로부터 다운로드 받고 디바이스로 전송 가능
  - 수집된 사고 정보 데이터를 구글 맵을 이용하여 시각화
